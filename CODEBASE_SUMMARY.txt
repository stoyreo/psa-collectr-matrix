KEY FEATURES
============

1. DATA INGESTION
   - Reads PSA collection export CSV (19 cards)
   - Normalizes column names and data types
   - Converts dates (mm/dd/yyyy), costs (float), quantities
   - Handles null values ("-", "N/A", etc.)

2. MATCHING ENGINE
   - Generates canonical match keys: subject_setcode_cardnum_grade_variety
   - Normalizes card names and set codes (sv1v, sv2a, m-p-promo, etc.)
   - Extracts set codes from complex names
   - Builds optimized eBay search queries

3. eBay COMPS FETCHING
   - Generates realistic sample comps in demo mode (12 per card)
   - Uses PSA Estimate as baseline pricing
   - Applies variance, grade multipliers, realistic dates
   - Filters bundles, fakes, damaged, wrong grades
   - Computes statistics: min, median, max, variance, trend
   - Flags outliers using IQR method (1.5x)

4. PRICE ANALYSIS
   - Market value = median of comps' total prices
   - Fallback to PSA Estimate when no comps
   - P&L calculation and percentage return
   - Confidence scoring 0-100 based on comp match quality

5. SIGNAL ENGINE
   - BUY: 15%+ upside + good liquidity + 85%+ confidence
   - SELL: 15%+ underwater OR declining trend
   - HOLD: stable with acceptable upside or illiquid
   - REVIEW: insufficient confidence or data
   - Risk assessment: LOW/MEDIUM/HIGH
   - Liquidity classification based on comp count & recency
   - Trend analysis (UP/DOWN/STABLE) from recent vs older comps

6. PORTFOLIO INSIGHTS
   - Top undervalued (highest upside %)
   - Top gainers (highest absolute position)
   - Weak positions (underwater cards)
   - High-risk holdings
   - Signal distribution
   - Liquidity distribution

7. CACHING
   - JSON-based caching (24-hour TTL)
   - Reduces repeated eBay searches
   - Metadata caching available

8. EXCEL EXPORT
   - 10 sheets: DASHBOARD, PORTFOLIO, EBAY_COMPS, COLLECTR_MAP, PSA_MAP,
     INSIGHTS, EXCEPTIONS, TARGETS, ALERT_LOG, CONFIG
   - Professional formatting:
     * Dark navy headers with white text
     * Alternating row colors (subtle gray/white)
     * Currency & percentage formatting
     * Conditional formatting (green/red for P&L)
     * Frozen header rows
     * Auto-column widths

PRODUCTION DEPLOYMENT
====================

To use in production (not demo mode):
1. Set DATA_SOURCES["demo_mode"] = False in config.py
2. Replace _generate_sample_comps() in ebay_connector.py with real
   eBay API calls or web scraping
3. Implement Collectr connector with browser automation (Selenium/Playwright)
4. Enable PSA cert verification API calls
5. Set up proper error handling for network issues
